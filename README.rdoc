= Redmine Select Watchers By Group Plugin

Allows issue creator to select multiple watchers by group

= Install

script/plugin install git://github.com/alvila/redmine_watchers_by_group.git

= Version Compatibility

This plugin is forked from the original one (alivila's) and maintained to support latest redmine versions.

Compatible with Redmine 2.0.x and 2.1.x.
